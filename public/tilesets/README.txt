Tileset structure for Tiled + this app

Drop your exported tileset PNG here:
public/tilesets/grassland.png

Recommended Tiled setup:
- Name: grassland
- Source: public/tilesets/grassland.png
- Tile width: 32
- Tile height: 32
- Margin: 0
- Spacing: 0

Optional:
- Keep original editable art files in public/tilesets/source/
- Keep alternate exported PNGs in public/tilesets/exports/

Current map file:
- public/maps/gym-run.tmj

If you use a different PNG filename, update the source path in Tiled to match it.
